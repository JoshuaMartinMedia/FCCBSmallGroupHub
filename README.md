# FCCB Small Groups Hub v7

A tiny Progressive Web App for small-group Bible reading schedules.

## v7 upgrades

- Added linking to Bible.com

## Uploading to GitHub Pages

Replace the existing files in your GitHub repository with these files, then commit changes.

If the app does not update immediately on your phone, close it fully and reopen, or refresh the browser page once.


## Bible.com links

Reading text is now clickable:
- Single chapter readings open that chapter.
- Same-chapter verse ranges open the exact verse range when possible.
- Multi-chapter readings open the first chapter in the assignment.
- Links use Bible.com NIV, version id 111.
